1ère partie : Installation des logiciels

0. se créer un compte sur https://github.com/ (designé par la suite par comptegithub)
1. allez sur le site http://www.ird.fr/us191/spip.php?article67
2. télécharger les logiciels "gitforwindows" et "tortoisegit"
   https://gitforwindows.org/
   https://tortoisegit.org/
3. Lancer l'installation de gitforwindows en tant qu'administrateur
4. lancer l'installation de tortoisegit en tant qu'administrateur

2ème partie : Configuration des logiciels, Initialisation d'un repository GitHub

1. lancer la console "Git Bash"
2. exécutez la commande $git config --global user.name "comptegithub"
3. exécutez la commande $git config --global user.mail "emailassociéaucomptegithub"
4. Connectez vous sur GitHub avec votre compte utilisateur
5. créez un nouveau repository "New repository", donner lui un nom (désigné par la suite par tutorialgithub" : le mettre en Public et une initialisation avec un README

3ème partie : Transférer un repertoire de votre PC sur le repository en ligne sur GitHub

1. Se placer dans la console Git Bash
2. exécutez la commande $cd chemindurepertoireàgéreravecgit (ici tutogit)
3. exécutez la commande $git init (création de .git)
4. git add -all
5. git remote add tutogit https://github.com/comptegithub/tutorialgithub.git
6. git commit -m "first commit"
7. git pull tutogit master
8. git push tutogit master
